
### 1. 開啟Anaconda Prompt
1. 創建環境，並安裝 Python 3.7
   ```
   conda create -n pyscpro python=3.7 -y
   ```
   
2. 啟動環境
   ```
   conda activate pyscpro
   ```
   ```
   conda install git
   ```
   
3.  Clone 專案
4. ```
   git clone https://github.com/danielpalen/pysc2-rl-agents.git
   ```
   ```
   cd pysc2-rl-agents
   ```

     
4. 安裝依賴套件(未成功)
   
   專案的主要依賴列在 `requirements.txt` 中
   
   ```
   pip install -r requirements.txt
   ```
